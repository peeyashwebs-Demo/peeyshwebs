# L - Hub Blueprint

## Overview
L - Hub is a premium, minimalist platform for creating, sharing, and discovering concise, media-rich tutorials. It focuses on clarity and learning through short videos and step-by-step images.

## Project Details & Features
- **Framework:** Next.js (App Router)
- **Backend:** Firebase (Auth, Firestore, Storage)
- **Styling:** Tailwind CSS (preferred for modern design) with a focus on premium aesthetics.
- **Themes:** Light and "Light Black" (Dim Mode).
- **Core Features:**
    - User Authentication (Email/Password, Social)
    - Home Feed with discovery and search.
    - Tutorial Creation (Media upload + Metadata).
    - Advanced Tutorial Player/Viewer.
    - User Profiles and Interactions (Likes, Comments).
    - Responsive Design.
